Cube version1

Sep 30, 2018

정육면체를 그려보자. 프로젝트명 Cube.

정육면체를 그리는데 이렇게 힘들 줄이야!

네모 6개를 그리고 transform을 이용해 네모를 변형하여 연결하면 정육면체를 그릴 수 있다고 생각했지만, 도형을 원하는 대로 변형하기 너무 어려웠다.
“정육면체 그리는 법”을 검색하지 않고 transform 관련 검색을 통해 네모를 변형하는 작업을 진행하고 싶었으나 검색 도중에 완성된 글을 보고 말았다.

Using CSS transforms
글을 읽어보아도 무슨 말인지 어떻게 화면에 이렇게 표현되는지 이해하기 어려워 perspective에 대해 찾아보았다

CSS3 3D effect(perspective) #1
친절한 설명으로 perspective가 원근감을 주는 초점인 것을 알게 되었다

perspective-origin, perspective, translateZ 등 값을 조절하여 위치와 모양을 변형해 보았다

front 네모와 back 네모가 각각 Z축이 translateZ(50px)로 설정되어있다

back의 Z축 설정을 -50px이 아니라 50px이라는 점이 신기했다
rotateY(180deg)로 기준 축의 방향이 바뀌어 -50px이 아니라 50px으로 Z축을 이동해야 뒤로 가는 효과가 나온다.
back에 쓰인 글자(숫자6)를 보면 방향이 바뀌었음을 알 수 있다.

도형을 변형하기 위해 찾아보던 도중 transform의 속성으로 translate, rotate, skew이외에 sknew가 있는 것을 이번에 알게 되었다.

backface-visibility: visible과 hidden의 차이를 제대로 아는 것인지 잘 모르겠다
CSS3 : backface-visibility 속성(image flip 구현)
이 설명을 읽고 난 후 차이점이 보여 알았다고 생각했지만
backface-visibility
이 글의 예제에서는 아무 차이가 없어 보여 이해가 덜된 것 같다.
